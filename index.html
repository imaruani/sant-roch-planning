Comment utiliser l'interface admin
▼
1
Créer les séances
Allez dans SÉANCES → + Nouvelle Séance.
Choisissez le type (Guidée / Freeflow), la date, l'heure et la durée.
Pour une séance guidée, un guide est obligatoire.
Utilisez Copier Semaine pour dupliquer un programme existant.
2
Consulter les dispos
Allez dans DISPONIBILITÉS.
Le fond de chaque créneau indique combien de flows sont disponibles :
■ rouge = 0 flow · ■ orange = 1 · ■ vert = 2+
Les points colorés = les flows disponibles sur ce créneau.
3
Assigner les flows
Cliquez sur un bloc séance dans la grille pour ouvrir le panneau d'assignation.
Vous voyez les flows disponibles, triés par équité (moins de séances ce mois d'abord).
Cliquez ASSIGNER — le flow reçoit un email et doit confirmer.
4
Auto-assigner
Le bouton Auto-Assigner assigne automatiquement les flows disponibles sur chaque créneau libre, en respectant l'équité.
Vous pouvez ensuite modifier manuellement chaque assignation.
5
Publier le planning
Une fois les assignations prêtes, cliquez Publier depuis l'onglet Séances.
Chaque flow reçoit un email avec ses séances de la semaine.
Ils doivent confirmer leur présence depuis leur espace.
6
Suivre les confirmations
Dans la grille des disponibilités : ✓ vert = confirmé, ... = en attente.
Dans PARAMÈTRES → Statut des confirmations : vue globale de toutes les séances à venir.
Astuce : Personnalisez les couleurs de chaque guide pour les retrouver facilement sur le calendrier.

? GUIDE & FAQ

C'EST QUOI UN SHIFT ?

Un shift est un créneau de présence à Sant Roch. Il peut contenir une ou plusieurs séances :

  • ️ Rituel libre — les membres explorent chaleur et froid à leur rythme, tu es là pour les accueillir et accompagner
  • Rituel guidé — session collective structurée, animée avec un guide

Exemple : shift 10h–15h30 = rituel libre 10h–12h + rituel guidé 13h–15h30

Comment déclarer mes dispos ? ▾

Va dans l'onglet MES DISPOS. Pour chaque shift de la semaine, tape sur le créneau :

  • ✓ Dispo → tu es disponible sur tout le créneau
  • ✗ Pas dispo → tu n'es pas disponible. Tu peux ajouter une note pour préciser tes contraintes

Si tu appuies à nouveau sur "✓ Dispo", ça réinitialise ta réponse.

⭐ Je suis dispo SEULEMENT sur une partie du shift ▾

Exemple : le shift est 10h–15h mais tu n'es dispo que de 10h à 13h.

  1. Appuie sur le bouton ✗ Pas dispo
  2. Un nouveau bouton apparaît : 📝 Laisser une note à l'admin — tape dessus
  3. Renseigne tes vraies heures de dispo dans les champs De / À (ex : De 10:00 — À 13:00)
  4. Ajoute un commentaire si tu veux (optionnel)
  5. Tape ENREGISTRER
💡 L'admin verra : "Pas dispo" + ta note avec tes vraies heures.
Il pourra couper le shift en deux (ex : 10–13h pour toi, 13–15h pour quelqu'un d'autre).
Comment confirmer un shift ? ▾

Quand l'admin te confirme sur un shift, tu reçois une notification dans MON PLANNING. Le shift apparaît en orange avec le bouton :

✓ CONFIRMER MON SHIFT

Appuie dessus pour valider ta présence. Il passe ensuite en vert ✓.

Où voir mon planning complet ? ▾

Dans l'onglet MON PLANNING — tu vois tous tes shifts assignés et rituels à venir. Tu peux basculer entre vue semaine et vue mois.

L'onglet SANT-ROCH montre le planning global de toute l'équipe.

Je n'ai pas reçu de notification ▾

Les notifications apparaissent en haut de l'onglet MON PLANNING à chaque ouverture de l'app. Pense à rafraîchir la page si tu ne les vois pas.
